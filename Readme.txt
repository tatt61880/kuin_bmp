BMPファイルから画像情報を読み込んだり、画像をBMPファイルとして保存するためのライブラリです。

免責事項:
    自己責任でご利用ください。

ファイルの説明:
    Bmp.kn
        ライブラリ本体です。

    BmpTest.kn
        Bmpライブラリの使用例です。
        exeファイルと同じディレクトリに画像ファイル(image.bmp)を置いて使用します。
        ※画像ファイル(image.bmp)が存在しなければ、真っ黒な画面を表示します。

    image.bmp
        BmpTest.exe で使用する画像ファイルです。
        中身は自由に編集して構いません。
